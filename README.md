📖 About
thpt-thanhmienI-reborn is a from-scratch redesign of the official website for THPT Thanh Miên I — a high school in Hải Phòng, Vietnam. The goal was simple: take a website that wasn't winning any design awards and rebuild it into something clean, readable, and actually pleasant to use.
Built with pure HTML, this project is a personal exercise in frontend development and a love letter (of sorts) to the school.
---
✨ Features
🏠 Home page — Clean landing page with school info and highlights
📰 News page — Browse the latest school news and announcements
📬 Contact page — Easy way to get in touch with the school
🖼️ Image assets — Organized media folder for all visuals
📱 Improved layout — A much more readable and modern design than the original

🚀 Getting Started
No build tools or dependencies needed — just plain HTML!
Clone the repository
```bash
   git clone https://github.com/thhanns/thpt-thanhmienI-reborn.git
   ```
Open in your browser
```bash
   cd thpt-thanhmienI-reborn
   open index.html
   ```
Or simply double-click `index.html` to open it in your browser.
